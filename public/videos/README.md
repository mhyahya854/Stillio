# Local Video Ambience Assets

Place your local video files in this directory.

## Optional Paths

There are no required video files for the default workspace state. If you want a scene to use motion instead of a still image, point that scene's `videoSrc` at a local file in this directory.

## Requirements:
- Format: `.mp4`
- Optimization: Recommended to be under 10MB for fast local loading.
- Loop: Videos should be seamlessly loopable.
- Audio: Audio is not used by the visual ambience system; it can be included but will be muted by the app.
